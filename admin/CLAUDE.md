# SecureGuard Admin Panel

Panel administrativo para gestión de la plataforma SecureGuard.

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Supabase (autenticación y datos)
- Recharts (gráficas)

## Secciones
- `/verificacion` — Cola de agentes pendientes de verificación
- `/metricas` — KPIs y estadísticas en tiempo real
- `/disputas` — Gestión de disputas entre clientes y agentes
- `/finanzas` — Pagos, payouts y reportes financieros
- `/agentes` — CRUD de agentes verificados
- `/clientes` — CRUD de clientes
- `/configuracion` — Configuración de la plataforma
